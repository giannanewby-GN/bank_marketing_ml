# Bank Marketing Prediction Model (Logistic Regression & Decision Tree)

This project analyzes a real-world bank marketing dataset to predict whether a customer will subscribe to a term deposit. Using Python, Pandas, and scikit-learn, two models—Logistic Regression and Decision Tree—were trained and compared to determine which approach performs better on classification tasks.

## Project Goals
- Explore a real marketing dataset
- Clean and preprocess input variables
- Build predictive models that classify subscription outcomes
- Compare performance metrics across models
- Interpret model insights to understand customer behavior patterns

## Tools & Technologies
- **Python** (Pandas, NumPy, scikit-learn)
- **Jupyter Notebook**
- **Matplotlib / Seaborn**
- **Classification Models:** Logistic Regression, Decision Tree

## Files in this Repository
- `BankMarketing.ipynb` — main notebook with code, exploration, model training, and evaluation
- `report.pdf` — (optional) summary of findings and model insights

## Dataset
This project uses the **Bank Marketing dataset** from the UCI Machine Learning Repository.

Dataset Link:  
https://archive.ics.uci.edu/ml/datasets/bank+marketing

Note: The dataset is publicly available, so it is not included directly in this repository.

## Methods & Workflow
1. **Loaded and inspected data**  
2. **Performed data cleaning and feature engineering**  
3. **Trained two classification models:**  
   - Logistic Regression  
   - Decision Tree  
4. **Evaluated models using:**  
   - Accuracy  
   - Precision & recall  
   - Confusion matrix  
5. **Compared results and interpreted key predictors**

## Key Results
- Logistic Regression provided more interpretable coefficients.
- Decision Tree offered better decision boundaries in some cases.
- Subscription likelihood was strongly influenced by factors such as age, education, housing loan status, and contact method.
